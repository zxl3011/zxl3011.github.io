# Lucy Zhang Portfolio

This repository contains my personal portfolio website, built with Astro and deployed with GitHub Pages.

Live site: https://zxl3011.github.io/

## About

This portfolio showcases my software development projects, technical skills, and learning notes. It is designed to support my applications for graduate and junior software developer roles in Australia.

## Tech Stack

- Astro
- TypeScript
- HTML / CSS
- GitHub Pages
- GitHub Actions

## Featured Projects

- Surfboard Rental Management System
- Indigenous Art Atlas
- Enterprise Operations Portal
- Smart Parking Chatbot
- Workforce Time Management System

## Deployment

The site is automatically deployed to GitHub Pages using GitHub Actions when changes are pushed to the `main` branch.
